A pasta src é composta por duas subpastas. Uma pasta server e outra pasta player

Para correr a aplicação basta, compilar os executáveis usando o comando "make" nesta diretoria,
de seguida basta correr o servidor usando o comando "./server", e por fim correr o cliente usando o comando "./player".

Observações:
 - Os ficheiros recebido como resposta aos comandos show_trials e scoreboard são guardados na pasta src/player/scores, 
 em relação à diretoria atual
 - Todos os path são tratodos de forma dinâmica em relação à diretoria principal, portanto se as pastas
server e user forem mudadas de localização estes path terão de ser alterados
